# Milestone Plan

## Milestone 01 - Autonomous Repo Operating System and Truth-Layer Foundations
- Audit repo and choose stack.
- Create AGENTS hierarchy, memory system, skills, architecture docs, progress logs, review docs, and PR draft flow.
- Scaffold monorepo tooling and quality gates.
- Define canonical domain package and worker/service boundaries.
- Establish initial ingestion/orchestration/API/web/mobile shells.

## Milestone 02 - Schema + Persistence Backbone
- Add concrete database package and forward-only migrations.
- Implement staging, normalized, validation, and publish persistence.
- Add replay tooling and migration safety checks.

## Milestone 03 - Provider Integration and Validation Depth
- Implement at least one real provider adapter behind interfaces.
- Add entity resolution, normalization rules, and validation packs.
- Stand up scheduled orchestration and freshness monitoring.

## Milestone 04 - Read APIs and Search Foundations
- Expose typed player/team/match/season endpoints.
- Add search/index contracts and ingestion into search backend.
- Add auth and rate-limiting groundwork for B2B APIs.

## Milestone 05 - Product Surfaces and Narrative Layer
- Build mobile-first product flows for search, entity views, comparison, trends, and narrative explanations.
- Add editorial/scouting modules backed by published truth.
